# acapy-issuer-ui



## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## Routing for Development
```
ssh -R 10000:localhost:10000 user@ssh-server
```
